What is Bootstrap

Bootstrap is the most popular framework for building responsive, mobile-first sites, with BootstrapCDN and a template starter page.

Many components in bootstrap require the use of JavaScript to function. Specifically, they require jQuery, Popper.js, and JavaScript plugins build by bootstrap.

You need place three scripts near the end of your pages, right before the closing body tag to make the frame work. Those scripts must follow the order which is jQuery first, then Popper.js, and then the JavaScript plugins.

Benefit with Bootstrap

Responsiveness

Creating mobile-first websites with Bootstrap is very easy, with the Bootstrap responsive grid easily adapting your website layout to display correctly on common screen sizes without create css file.

Learn more

Improve efficiency

Bootstrap will significantly speed up development time and improve workflows. And the bootstrap will help you avoid common styling mistakes like centring text, floating elements and changing font sizes with built-in classes automaticly.

Learn more

Versatility

Bootstrap supports the multiple versions of browser include Google Chrome, Firefox, Internet Explorer, Opera, and Safari. It even provides support as far back as Internet Explorer 8, meaning Bootstrap will offer you wider accessibility right from the start.
